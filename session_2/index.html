    매번 설명하지 말고, Project Instructions에 적어두면 계속 적용됩니다

02
병렬 실습 ①
복붙 지시문 → 혼자 시도 → 순회 코칭
⏱ 50분
실습 운영 방식
3박자로 굴러갑니다
Step 1

복붙 지시문

준비된 지시문 카드를 그대로 복사 → 붙여넣기. 빈칸 [ ]만 내 정보로 채웁니다

Step 2

혼자 5분 시도

Run 실행 후 결과를 직접 봅니다. 마음에 안 들면 한 줄씩 고쳐봅니다

Step 3

순회 코칭

강사·보조강사가 트랙을 돌며 막힌 지점을 함께 풉니다

지시문 카드는 Slack에 미리 공유되어 있습니다. 화면을 따라 적지 말고 복사해서 쓰세요.
🎬 Track A · 콘텐츠 / 크리에이티브
복붙 실습 지시문
A-1 · 채널별 카피 변주 (Chat)
다음 메시지를 채널별로 변주해줘. - 인스타: 감성·150자, 해시태그 별도 줄 - 유튜브 커뮤니티: 정보형·300자 - 틱톡: 첫 줄 후킹·80자 원본 메시지: [붙여넣기]
A-2 · 폴더 톤 학습 → 보도자료·디스크립션 (Cowork)
이 폴더의 과거 보도자료(또는 상품 디스크립션)를 먼저 읽고 톤·형식을 파악한 다음, 같은 스타일로 [신작/신상품] 초안을 써줘. 사실관계가 불확실한 부분은 [확인 필요] 표시.
A-3 · 내 브랜드 Project 지침화 (Chat)
앞으로 내 모든 작업에 적용할 지침을 만들어줘. - 브랜드 톤: [예: 차분·시적, 형용사 남발 금지] - 금지어: [ ] - 기본 출력 형식: [예: 표 / 불릿 3개] 이걸 Project Instructions에 넣을 수 있게 정리해줘.
📊 Track B · 세일즈 / 영업 · 문서자동화
복붙 실습 지시문 — 딱 2개만, 결과물 1개 완성이 목표
B-1 · 세일즈 교육자료 / 브리핑 1장 (Chat → Artifacts)
너는 10년차 영업 팀장이야. 신입 대상 5분짜리 교육자료를 한 장으로 만들어줘. 구성: ① 오늘의 목표 1줄 ② 핵심 멘트(스크립트) 3개 ③ 현장 체크리스트 5개 주제: [예: 첫 고객 응대 / 매물 브리핑 / 상품 소개] 결과는 Artifacts로 보여줘.
B-2 · 폴더 자료 읽고 문서 자동 생성 (Cowork)
이 폴더의 과거 [뉴스레터 / 브리핑]을 모두 읽고 톤과 형식을 파악한 다음, 같은 스타일로 이번 회차 초안을 써줘. - 새로운 내용: [붙여넣기] - 추측한 부분은 [확인 필요] 표시 - 결과를 폴더에 파일로 저장해줘
💡 욕심내지 않기. 2개 중 1개라도 끝까지 완성하면 오늘은 성공입니다.
📊 Track B · 여유 있는 분만
보너스 — 반자동 리스트 정제
B-3 · 잠재고객 리스트 정제 (Chat)
아래 뉴스·기업 정보 텍스트를 읽고, 잠재 고객사 리스트를 표로 정제해줘. 열: 회사명 / 추정 담당 부서 / 관심 포인트 / 근거가 된 기사 한 줄 불확실하면 [확인 필요]로 비워둬. 텍스트: [붙여넣기]
"완전 자동수집·자동발행"은 오늘 범위가 아닙니다. 뉴스 자동 수집 → 뉴스레터 자동 발행처럼 스케줄·웹검색·연동이 필요한 자동화는 3회차에서 다룹니다. 오늘은 "붙여넣기 → 정제"의 반자동까지.

Track A 포인트

"톤"을 고정

브랜드 보이스·금지어·문체를 지침에 박아두면, 매번 톤을 설명할 필요가 없어집니다

Track B 포인트

"출력 형식"을 고정

브리핑은 항상 같은 표, 교육자료는 항상 같은 3단 구성 — 형식을 지침에 넣어두기

🎯 신규자 목표는 "완벽한 지침"이 아닙니다. 한 줄짜리라도 직접 만들어본 경험이면 충분합니다.
